## mesg-cli workflow kill

Kill a workflow from the Network and get back its token

### Synopsis

Kill a workflow from the Network and get back its token

```
mesg-cli workflow kill WORKFLOW_ID [flags]
```

### Examples

```
mesg-cli workflow kill WORKFLOW_ID
mesg-cli workflow kill WORKFLOW_ID --account ACCOUNT --confirm
```

### Options

```
  -a, --account string   Account to use
  -c, --confirm          Confirm
  -h, --help             help for kill
```
