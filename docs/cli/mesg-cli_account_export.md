## mesg-cli account export

Export accounts in order to be able to re-import it with the import command

### Synopsis

Export accounts in order to be able to re-import it with the import command

```
mesg-cli account export [flags]
```

### Examples

```
mesg-cli account export
```

### Options

```
  -h, --help          help for export
  -n, --name string   Name of the account
```
