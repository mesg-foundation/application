## mesg-cli service list

List of services that a account already deployed on the Network

### Synopsis

List of services that a account already deployed on the Network

```
mesg-cli service list [flags]
```

### Examples

```
mesg-cli service list
```

### Options

```

  -h, --help             help for list
```
